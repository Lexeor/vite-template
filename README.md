# Vite + React + TypeScript + Tailwind 4 + Framer Motion Template

A premium, modern starter template for high-performance web applications.

## ✨ Features

- **Vite 7**
- **React 19**
- **TypeScript**
- **Tailwind CSS 4**
- **Framer Motion**
- **Lucide React**

## 🚀 Getting Started

### 1. Clone the template
```bash
# You can use this as a template or clone it
git clone <your-repo-url>
cd <your-project-name>
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```

## 📁 Project Structure

```text
src/
├── assets/      # Static assets
├── components/  # Reusable UI components
├── App.tsx      # Main application component
├── index.css    # Global styles (Tailwind 4 configuration)
└── main.tsx     # Application entry point
```
